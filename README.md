# API Users

## Descrição

Esta aplicação é uma API de Usuários que gerencia a criação, atualização, consulta e remoção de usuários.

## Recursos

- Cadastrar as usuários

## Pré-requisitos

- Node.js (v18 ou superior)
- pnpm (v9 ou superior)
- docker (v23 ou superior)
- docker compose (v1 ou superior)

## Instalação

1. Clone o repositório para o seu ambiente local:
    ```sh
    git clone https://github.com/joaouriel10/api-users.git
    cd api-users
    ```

2. Instale as dependências:
    ```sh
    pnpm install
    ```

## Execução

Para iniciar a aplicação em modo de desenvolvimento, utilize o comando:
```sh
docker compose up -d
pnpm run start:dev
